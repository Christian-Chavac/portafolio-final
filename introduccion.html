Proyecto: Desarrollo de Portafolio Web Personal.

Tipo de proyecto: Individual.

Descripción general

En el curso de Introducción a los Sistemas de Cómputo se desarrolló un proyecto individual enfocado en la creación de un portafolio web personal relacionado con las competencias adquiridas durante el primer semestre.

El sitio web está orientado a mostrar todos los proyectos desarrollados en los diferentes cursos del semestre mediante una página organizada y funcional.

El proyecto fue dividido en varios incrementos o etapas de desarrollo, iniciando con la investigación de portafolios profesionales, documentación de competencias, uso de Git, GitHub, hosting gratuito, HTML y CSS.

Objetivo del proyecto

Desarrollar un sitio web personal que permita mostrar de manera organizada las competencias y proyectos realizados durante el semestre, aplicando conocimientos básicos de desarrollo web y herramientas tecnológicas.

Aprendizajes obtenidos

Este proyecto permitió fortalecer conocimientos relacionados con HTML, CSS, Git, GitHub y hosting web.

También ayudó a mejorar habilidades de organización, diseño web y documentación de proyectos.
